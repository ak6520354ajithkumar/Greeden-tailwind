# 🌿 Greeden - Tailwind CSS Project

A modern and responsive plant store landing page built using **HTML** and **Tailwind CSS**. This project focuses on creating a clean, user-friendly UI with a mobile-first responsive design.

✨ Features

- 🌱 Responsive design
- 🎨 Modern and clean UI
- 📱 Mobile-first layout
- ⚡ Built with Tailwind CSS
- 🛒 Product showcase section
- 📞 Contact section
- 🧭 Responsive navigation bar

🛠️ Technologies Used

- HTML5
- Tailwind CSS
- CSS3


🎯 Learning Outcomes

Through this project, I learned:

- Tailwind CSS utility classes
- Responsive web design
- Flexbox and Grid layouts
- Mobile-first development
- Clean UI design principles
